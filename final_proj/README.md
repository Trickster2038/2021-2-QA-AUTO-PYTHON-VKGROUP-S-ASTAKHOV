# Финальный проект

## Api

- Удаление через GET-запрос

```
GET http://<APP_HOST>:<APP_PORT>/api/del_user/<username>
```

- Код 210 вместо 201
- Отсутствие валидации пустого логина, пустого пароля, невалидного или занятого email
